**数据锚点内容：**
| Provider | 锚点字段 | 来源 |
|----------|---------|------|
| Indicator | `decision_tf_bias`, `alignment`, `conflict_count`, `lower/higher_tf_agreement` | `BuildIndicatorStateJSON` cross_tf_summary |
| Structure | `latest_break_type`, `latest_break_bar_age`, `supertrend_state/interval` | TrendJSON 解析 |
| Mechanics | `conflicts[]`, `reversal_risk` | `BuildMechanicsStateSummary` |

**设计考量：**
- 锚点仅用于**交叉验证**，prompt 约束明确说明"不作为独立判断依据"
- 每个 Provider 仅接收与其领域相关的锚点（token 效率）
- 空锚点（nil/空对象）不注入 payload，避免无效 token 消耗

### ✅ Task 9+10: 新增 5 个技术指标

**新增文件：**
- `internal/decision/features/indicator_extra.go` (~260 行)
  - `computeBollingerBands` — SMA(20) ± 2×StdDev，输出 %B + width
  - `computeCHOP` — Choppiness Index (period=14)，38.2/61.8 阈值区分趋势/震荡
  - `computeStochRSI` — Stochastic RSI (period=14)，从 RSI 序列计算
  - `computeAroon` — Aroon Up/Down (period=25)
  - `computeTDSequential` — TD 序列 setup 计数器
  - 每个指标含 snapshot 结构体和 build 函数

**修改文件：**
- `internal/decision/features/indicator_compress.go`
  - `indicatorData` 新增 5 个 snapshot 字段
  - `IndicatorCompressOptions` 新增 BBPeriod=20, BBMultiplier=2.0, CHOPPeriod=14, StochRSIPeriod=14, AroonPeriod=25
  - `buildIndicatorData` 接线所有新指标（StochRSI 依赖 RSI 序列，在 RSI block 内计算）

- `internal/decision/features/indicator_state.go`
  - `indicatorMomentumState` 新增 StochRSIZone, AroonSignal
  - `indicatorVolatilityState` 新增 BBZone, BBWidthState, CHOPRegime
  - 5 个新分类函数：`classifyBBZone`, `classifyBBWidthState`, `classifyCHOPRegime`, `classifyStochRSIZone`, `classifyAroonSignal`
  - `detectIndicatorEvents` 新增 TD Sequential 事件检测

**分类阈值（量化工程师视角，BTC/ETH 优化）：**
| 指标 | 阈值 | 依据 |
|------|------|------|
| BB %B | <0 below_lower, 0-0.2 near_lower, 0.2-0.8 mid, 0.8-1.0 near_upper, >1.0 above_upper | 标准 BB 区间 |
| BB Width | <2% squeeze, >6% wide | 15m-4h 加密货币波动率 |
| CHOP | <38.2 trending, >61.8 choppy | Fibonacci 水平（标准实践） |
| StochRSI | ≤0.2 oversold, ≥0.8 overbought | 标准随机 RSI 阈值 |
| Aroon | Up>70&Down<30 strong_up, 反向 strong_down, 均>70 crossover | 标准 Aroon 信号解释 |
